⚡ Geolocation Timeout Test v4

Pure timeout measurement — no cache warm-up, no watch, no noise. Tests whether getCurrentPosition resolves within the given thresholds.

How it works: For each configured timeout value, a fresh getCurrentPosition call is made with maximumAge: 0 and that timeout. If the call times out → TIMEOUT (real GPS can't respond that fast). If it resolves → INSTANT (suspiciously fast for a cold call). All timeout calls fire in parallel per run.
